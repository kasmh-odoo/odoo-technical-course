# Eagle Estates 🦅

A real estate management agency has approached your company to create a software solution that will help them manage their properties.  
The agency has a large number of properties, and they need a system that can help them manage their listed properties and linked rooms.

Here is an overview of the user specifications:

- Manage a database of all the properties 🏘️
- For each property, manage the following:
    - Rooms 🛏️
    - Apartments/Units 🏠
    - Compounds/Buildings 🏢
- Create a printable report that provides a property's details 🖨️
- Display properties and their details on an eCommerce platform 🌐

Based on your knowledge of Odoo's framework, implement the exercises in this case study using the best practices. If you feel unsure at any time, you can refer to the solutions.

All the best! 😉
