# Eagle Estates

A real estate management agency has approached your company to create a software solution that will help them manage their properties. 
The agency has a large number of properties, and they need a system that can help them manage their listed properties and linked rooms.

These are their requirements:
    -   Manage a repository of all of their Properties
    -   For each property, allow management of:
        -   Rooms
        -   Apartments/Units
        -   Compounds/Buildings
    -   Create a printable report that lists a property's details: Property Sheet
    -   Expose a public frontend for visitors/tenants to view a list of published properties
    -   ...etc.
